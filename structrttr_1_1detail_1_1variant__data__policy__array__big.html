Detailed Description

template<typename T>
struct rttr::detail::variant_data_policy_array_big< T >

This policy is used for raw array types of arbitrary rank, which fit NOT into variant_data.

A copy of the given array value will be allocated on the heap. The pointer to the array is stored in variant_data.
